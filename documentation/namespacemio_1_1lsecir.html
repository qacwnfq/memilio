Performs a simulation with specified parameters for given model.

Parameters
[in]t0Start time of the simulation.
[in]tmaxEnd time of the simulation.
[in]dtInitial step size of integration.
[in]modelAn instance of a LCT model for which the simulation should be performed.
[in]integratorAn integrator that should be used to discretize the model equations. If default value is used the simulation will be performed with the runge_kutta_cash_karp54 method.
Returns
A TimeSeries with the result of the simulation.
