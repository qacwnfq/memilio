template<class MultiIndex >
mio::IndexRange< MultiIndex >::IndexRange ( const MultiIndex &  dimensions)

Construct a Range that can be used to iterate over all MultiIndices in the given dimensions.

The range for each Index i in the MultiIndex is determined by [0, d_i), where d_i is the dimension of i.

[in]dimensionsA MultiIndex that contains the dimension for each Category.
